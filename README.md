# node-server-demo
Node跨域 Node.js部署 Vercel 使用 node.js 简单搭建Web服务/使用node简单搭建后端服务/使用node搭建服务/部署Vercel 部署vercel/express模块化开发

## Clone代码
```cmd
git clone https://github.com/poboll/node-server-demo.git
```

## 安装依赖

```cmd
npm install
```

## 启动

```cmd
node app.js
```


或者

```cmd
npm start
```
